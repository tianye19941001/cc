牡丹生态基地旅游 Peony Garden Ecotourism

菏泽市位于鲁西南平原，苏鲁豫皖四省八县交界处，素有“中原锁匙，四省通衢”之称，是发展牡丹深加工和商贸流通业的风水宝地。菏泽牡丹久负盛名，是著名的“牡丹之乡”、“中国牡丹城”。菏泽牡丹在国内、国际市场上都有很高的知名度和较强的市场竞争力，菏泽牡丹种植面积十余万亩，其种植面积、产量均居全国之首。品牌孵化方面，从科学种植到资源采收、从储存到各加工工艺链条，让消费者及各合作商通过旅游式参观，了解高品质牡丹产品的全透明物理生产过程。以各种形式对牡丹文化进行展览展示。以牡丹主题元素的休闲、娱乐、住宿、餐饮、运动等配套设施，来提高园区牡丹文化气息，以带动整个产业链的快速发展。

旅游路线

Travel ways
  • +

    “醉美”恩施

    “一山有四季，十里不同天”。以“雄、奇、秀、幽、险”自然风光著称，自然景观有：恩施大峡谷、恩施土司城、大清江画廊、利川腾龙洞、土家女儿城等。
  • +

    “秀美”芦芽山

    国家4A级，山西省第二大旅游景区。 82万亩原始次森林、66万亩草原、500多种动植物资源、100多个景点
  • +

    “神奇村寨”井底村

    这是一个被绝壁悬崖所环抱，有碧湖飞瀑所点缀的世外桃源。位于山西太行山峡谷深处，山涧泉水四季流淌，奇峰怪石遍布山野。
  • +

    “海南侨乡”兴隆

    碧海连天远，琼崖尽是春。海南岛地处热带北缘，属热带季风气候，素来有天然大温室的美称，独特的气候环境适宜养生健体
  • +

    “匡庐奇秀甲天下”庐山

    世界地质公园，5A级旅游景区。庐山又称匡山、匡庐，地处江西省北部，中华文明的发祥地之一，东偎婺源鄱阳湖，南靠南昌滕王阁，西邻京九大通脉，北枕滔滔长江。
  • +

    “千年瓷都”景德镇

    景德镇市，位于江西省东北部。春秋时属楚东境，秦为九江郡番县地，汉属豫章郡鄱阳县，东晋称新平镇。享誉全世界，历史上是官窑之地。
  • +

    “水墨”婺源

    婺源县隶属江西省上饶市，古徽州一府六县之一，素有“书乡”、“茶乡”之称，是全国著名的文化与生态旅游县。
  • +

    “玉桂成林”桂林

    桂林，国家AAAAA级旅游景区，中国十大风景名胜之一，桂林山水是中国山水的代表
  • +

    “英雄城”南昌

    南昌之名源于“昌大南疆、南方昌盛”之意，简称 “洪”或“昌”，古称豫章、洪都，江西省省会，是中国最早的航空工业基地、重要的综合交通枢纽和光电产业基地，世界级的光伏产业基地。
  • +

    “中国温泉之城”丹东

    丹东，位于辽宁省东南部，地处祖国边陲，历朝都是军事要塞。唐尧属青州，虞舜属营州。战国时为燕国东部边疆。
  • +

    “春城”昆明

    云南，素以其美丽、丰饶、神奇而著称于世，这里有风花雪月的大理名胜，高原水域的绝美丽江，神奇的香格里拉—中甸，孔雀曼舞的西双版纳
